# Bon Voyage

[![](https://img.shields.io/travis/olgabot/bonvoyage.svg)](https://travis-ci.org/olgabot/bonvoyage)[![](https://img.shields.io/pypi/v/bonvoyage.svg)](https://pypi.python.org/pypi/bonvoyage)

## What is `Bon Voyage`?

Transform percentage-based units into a 2d space to evaluate changes in distribution with both magnitude and direction.

* Free software: BSD license
* Documentation: https://olgabot.github.io/bonvoyage

## Installation

To install this code, clone this github repository and use `pip` to install

    git clone git@github.com:olgabot/bonvoyage
    cd bonvoyage
    pip install .  # The "." means "install *this*, the folder where I am now"
